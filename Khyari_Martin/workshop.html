Workshop

Le workshop avait pour objectif l'identification d'une problématique et la résolution de cette dernière.

Notre problématique fut la suivante : "Comment sécuriser l'utilisation d'internet pour les étudiants ?".

Pour répondre a cette problématique nous avons décider de mettre en place un gestionnaire de mots de passe qui permettrait a leur utilisateurs de stocké leur mots de passe dans une base de donnée sécurisée derrière un mot de passe unique.

Le processus de création fut dans un premier temps de créer une interface avec laquelle les utilisateurs pourrait se connecter a leur gestionnnaire de mots de passe. Pour cette interface nous avons décider de créer un site web en HTML CSS.

En ce qui concerne la base de données nous nous sommes servi du logiciel XAMPP pour PHPMyAdmin

Ce projet nommé Safe Guard Pass ne peut etre que entamé du a la trop courte marge de réalisation néanmoins cela nous a permit de nous initier a la conception et gestion de base de données.
